* r: 피봇값보다 작은 값, 혹은 l 과 교차할때 까지 이동
  * 왜? l 과 r 이 교차하면 r은 피봇을 기준으로 큰 값과 작은 값의 사이에 위치함
* l 과 r 이 교차하면 lst[r] 과 피봇의 위치를 바꿈
  * 해당 피봇에 대해 정렬이 끝
